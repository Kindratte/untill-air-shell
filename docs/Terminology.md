# Terminology

![](screen_2.jpg) 

1. `Untill Air Shell` or just `Shell` - managment tool for `Untill Air POS` application, that allows users to:
    - Manage Back-office data
    - Generate reports
    - Manage application
    - Manage tasks
    - Use different custom `Views` to achive needed functionality.
1. `Views Container` - main navigation block of `Shell`
1. `View` - `Shell` plugin that provides some functionality and instruments to iteract with back office database using `Shell API`.
1. `Location` - subsection of `View`, that separate View to a different logical sections.
1. `Workspace` - displays content of current `View`
<br>
<br>
**`Shell API`** - a set of JavaScript methods, available to each `View`, that provides all necessary instruments to itteract with backoffice database.
